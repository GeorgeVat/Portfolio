Projects Timeline

Text Classification Using Logistic Regression

In this project, we investigate the automation of training algorithms in automated detection for extreme texts and users, with the use of supervised learning. For training properly the models, there are used 2 data sources from t wo different platforms. The first source was collected from the Parler platform, where it was used to “catch” the meaning of the far-right text, and the second was collected from the subreddit politics from Reddit, where the content of its posts are politically correct.

10/2021 - 03/2022

Exploratory Data Analysis on Parler Social Media Platform

This project focuses on studying the features of the social network Parler by analyzing the content of the users,the hashtags and their popularity until the platform was blocked. The dataset covers the period from October 1, 2020, to November 1, 2021, which coincides with the American elections. Given Parler's known pro-Trump user base, it is expected to observe a significant amount of discussion and excitement related to Trump and the elections during this period.

02/2022 - 02/2022

Yelp Business Dataset User Similarities

This project focuses on utilizing UCF (User-User Collaborative Filtering), ICF (Item-Item Collaborative Filtering), and Singular Value Decomposition (SVD) techniques to identify similar users. The aim is to determine the most suitable algorithm and optimal value for k (number of similar users) in order to improve predictions..

02/2021 - 03/2021

Yelp Business Dataset Clustering

This project focuses on utilizing k-means and agglomerative clustering algorithms to classify businesses based on their reviews. By applying these algorithms, the study aims to identify patterns and similarities in business reviews to assign them to relevant categories.

12/2020 - 01/2021

Spatial Data Retrieval

In the context of this work, the aim is to develop indexing techniques and queries for spatial data. This project implements the sort-tile-recursive (STR) technique. It reads rectangles from a file and constructs an R-tree in memory using this technique. Once the R-tree is built, range queries are implemented to perform queries on the spatial data.
